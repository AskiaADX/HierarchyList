Levels show details >>

Levels specify the fields which contains the unique identifier and the name of each level.

For example, if you have a CSV with the following fields:

"brand_id", "brand", "model_id", "model", "version_id", "version"

You can then specify each brand / model / version using:

  • Level 1: id = brand_id, name = brand
  • Level 2: id = model_id, name = model
  • Level 3: id = version_id, name = version
Live values

Indicates the fields value you want to save during the selection at each level.
Using the `live values` the respondent could partially select information without to fill each levels.
The last level will save all the field specified in the `Values to save` property of the ADC.
Enter each name of fields separate with coma.

Related to the above example, respondent could only select `brand` and `model` without to select `version`.
In that case speecified the following information:

  • Level 1: id = brand_id, name = brand, live values = brand_id
  • Level 2: id = model_id, name = model, live values = model_id
